Comment installer Win10 Notifications :

1. Copiez les documents

Donc, après avoir téléchargé les fichiers, copiez les fichiers

winnotif.js et winnotif.css qui sont inclus.

2. Importez les fichiers

Mettez ces deux lignes dans le code HTML de votre page Web pour importer Win10Notifications

Code d'importation HTML :

<link rel="stylesheet" href="winnotif.css">
<script type="text/javascript" src="winnotif.js"></script>

3. Mettez en place le code

C'est plutot facile puisque vous n'avez qu'à copier cet autre bout de code que vous mettrez dans la balise <body> de votre page Web.

Code de mise en place HTML

<div id="not">
<p id="title"></p>
<p id="body" style="font-size: 16px;"></p>
</div>

4. Utilisez le

Vous avez terminé ! (C'était pas ci difficile). Pour afficher la notification, vous devez faire exécuter la méthode notif() qui fait tout. Vous avez ci-dessous des exemples ainsi que des boutons pour tester les trpos exemples.

Code JavaScript de Win10 notifications

//Ok, donc, pour afficher une notification simple, vous utilisez ça :
notif("It's a notif");

//Pour mettre une description dans la notification, mettez simplement ce code :
notif("Title","Put some body...");

//Pour finir, vous pouvez changer le délai de désapparition... Il est par défaut de 10 secondes
notif("Title","Put some body...",1); //Note: la durée est exprimée en secondes
